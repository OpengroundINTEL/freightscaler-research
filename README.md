# freightscaler-research

Research package for climate, insurance, and freight logistics.

## Repository Layout

- `program/`: Python code for synthetic data generation and freight climate risk scoring
- `program_output/`: generated CSV and JSON outputs from the program
- `outputs/integration/`: integrated dissertation drafts and the disagreement log

## Key Dissertation Artifact

The main dissertation draft is in `outputs/integration/full-dissertation.md`.
